## 📄 研究论文

- **SkillOpt: 首个面向 Agent 技能的系统性文本空间优化器** 📅 2026-05-25 | arXiv:2605.23904
  作者/机构：Yifan Yang, Ziyang Gong 等（微软亚洲研究院等）
  核心贡献：提出了 SkillOpt，首个将 Agent 技能文档视为"冻结模型的外部状态"进行系统性优化的框架，引入文本学习率、拒绝编辑缓冲区和 epoch 级更新机制。
  摘要：当前 Agent 技能主要依赖手工编写或一次性生成，缺乏像深度学习优化器那样的可复现训练流程。SkillOpt 使用独立的优化器模型，将带评分的 rollout 转化为对技能文档的有界增/删/替换编辑，仅在验证分数严格提升时才接受编辑。在 6 个基准、7 个目标模型、3 种执行框架（直接聊天、Codex、Claude Code）的 52 个评估单元中，SkillOpt 全部取得最佳或并列最佳。在 GPT-5.5 上，直接聊天准确率提升 23.5 个百分点，Codex agentic 循环中提升 24.8 个百分点。迁移实验表明优化后的技能可跨模型规模和执行环境保持价值。
  链接：[arXiv](https://arxiv.org/abs/2605.23904)

- **MemAudit: 面向 LLM Agent 记忆投毒的事后因果审计框架** 📅 2026-05-25 | arXiv:2605.23723
  作者/机构：Zhewen Tan, Yilun Yao 等
  核心贡献：提出 MemAudit 框架，结合反事实记忆影响力评分和记忆一致性图谱，精准定位 Agent 记忆中导致有害行为的恶意记录。
  摘要：大语言模型 Agent 越来越依赖持久化记忆来存储交互历史并改善长期任务执行，但这也创造了安全漏洞——对抗性用户可通过普通交互注入恶意记忆记录，进而引导 Agent 的推理和行为。现有防御主要关注在线干预，但未解决"事后追溯哪些记忆导致了有害行为"的问题。MemAudit 结合两种互补信号：(1) 反事实记忆影响力评分，衡量每条记忆对有害输出的因果贡献；(2) 记忆一致性图谱，识别记忆库中的结构异常记忆。评估结果显示，QA 攻击成功率从 70% 降至 0%，RAP 攻击成功率从 83.3% 降至 0%。
  链接：[arXiv](https://arxiv.org/abs/2605.23723)

- **Agentic Proving for Program Verification: Claude Code 在程序验证中达 98.1% 端到端成功率** 📅 2026-05-25 | arXiv:2605.23772
  作者/机构：Alessandro Sosso 等
  核心贡献：评估 Claude Code 在 agentic proving 框架下用于 CLEVER（Lean 4 程序验证基准）的表现，端到端生成与验证管道成功率达 98.1%。
  摘要：Agentic 系统近期成为自动化定理证明的前沿方法。本文评估 Claude Code 在 agentic proving 框架下用于程序验证的能力。结果显示 Claude 为 98.8% 的问题生成了有效规约，对正确规约的实现认证率达 87.5%，端到端程序生成与验证管道成功率 98.1%。Claude 还能提供高质量的自我反馈，识别失败原因和数据集中的残留 bug。这一结果表明现代 agentic prover 的能力已超越现有程序验证基准的难度，需要更严格的评估方法论。
