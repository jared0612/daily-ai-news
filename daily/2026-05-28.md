## 📄 研究论文

- **Calibrating Conservatism for Scalable Oversight（可扩展监督的校准保守主义）** 📅 2026-05-27 | arXiv:2605.28807
  作者/机构：William Overman
  核心贡献：提出 Calibrated Collective Oversight (CCO) 框架，通过聚合多样化的辅助评分函数来衡量与保守基线的偏差，灵感来自 Attainable Utility Preservation。CCO 使用 Conformal Decision Theory 在线校准保守程度，确保不良结果保持在用户指定的阈值以下，并提供有限时间界限保证。
  摘要：随着 Agentic AI 系统获得自主规划和长时间环境交互的能力，人类如何对这些可能超越自身能力的系统进行有意义的监督成为根本性挑战。现有可扩展监督方法依赖复杂假设，大多停留在启发式层面，缺乏顺序设置下的统计保证。CCO 框架引入集体保守主义机制：当监督者对某行动无异议时高效用行动仍可被选中，只有当担忧累积时才会被覆盖。在修改版 SWE-bench 上，较弱的监督者成功约束了对抗性错位的更强 Agent；在 MACHIAVELLI 上，CCO 大幅减少伦理违规同时保持奖励。
  链接：[arXiv](https://arxiv.org/abs/2605.28807)

- **Contrastive Reflection Enables Rapid Improvements in Reasoning（对比反思实现推理快速提升）** 📅 2026-05-27 | arXiv:2605.28742
  作者/机构：Linas Nasvytis
  核心贡献：提出 Contrastive Reflection (CORE) 算法，通过比较成功与失败的推理轨迹生成自然语言洞察，以非参数化方式实现模型快速自改进，仅需少量训练样本和更少的模型 rollout 即可超越 GRPO、GEPA 等方法。
  摘要：语言模型可以利用可验证奖励在多种推理任务上提升能力，但参数化（如 RLVR）和非参数化（如 prompt 优化）方法通常需要数百个训练样本和数千次模型 rollout。CORE 通过对比过去推理轨迹中的成功与失败案例生成洞察——即捕捉差异的简短自然语言描述。在四个推理任务上，CORE 实现了比参数化和非参数化方法更快的改进，且使用的 rollout 更少。在仅 5 个训练样本的固定 rollout 预算下，CORE 也达到了可比甚至更好的性能增益。CORE 还在上下文效率方面大幅优于非参数化基线。
  链接：[arXiv](https://arxiv.org/abs/2605.28742)

- **Are Search Agents Searching, or Just Verifying What They Already Know?（搜索 Agent 究竟在搜索还是在验证已知信息？）** 📅 2026-05-27 | arXiv:2605.28721
  作者/机构：Huimin Fan
  核心贡献：发现 LLM 搜索 Agent 存在"内在知识依赖"(IKD)——即使有工具访问权限，Agent 往往依赖模型内置知识而非外部证据。同时推出 LiveBrowseComp 深度搜索基准，所有模型在该基准上的准确率均低于 2%。
  摘要：研究发现 Agent 在 BrowseComp 上即使有工具也能回答高达 44.5% 的问题，超过一半的搜索查询来自内部假设而非检索到的线索，且在移除支持证据后表现不如闭卷基线。这表明静态搜索基准可能奖励记忆驱动的验证而非证据驱动的发现。LiveBrowseComp 包含 335 个人工撰写的问题，答案依赖基准构建前 90 天内发布的事实，过滤掉了全球性重大事件。在该基准上，所有 Agent 闭卷准确率低于 2%，搜索增强得分相比 BrowseComp 下降 25-40 个百分点，且此前的模型排名不再可靠地预测性能。
  链接：[arXiv](https://arxiv.org/abs/2605.28721)
