- **Automated Reproducibility Assessments in the Social and Behavioral Sciences Using LLMs** 📅 2026-06-11 | arXiv:2606.13670
  作者/机构：Stefan Feuerriegel 等
  核心贡献：首次系统性证明 LLM 可用于自动化社会科学可重复性评估；LLM 管线在定性结论匹配上达到 96%，超越人类重分析的 74%。
  摘要：社会科学和行为科学的可重复性通常由独立研究者重新分析原始数据来评估，但这种方式资源密集且难以扩展。本文展示了 LLM 可以自动化可重复性评估。使用 N=76 项具有预定义声明的已发表研究，将 LLM 生成的分析与原始发现和人类重分析进行比较。在剩余研究中，LLM 管线在 41% 的研究中恢复了原始效应量（Cohen's d ±0.05 容差），在 96% 的案例中达成了与原始研究相同的定性结论。相比之下，人类重分析师在 34% 的研究中恢复了原始效应量，在 74% 的案例中达成一致。结果表明 LLM 可作为可重复性评估的可扩展工具。
  链接：[arXiv](https://arxiv.org/abs/2606.13670)

- **Agentifying Agent Assessment for Openness, Standardization, and Reproducibility** 📅 2026-06-11 | arXiv:2606.13608

  摘要：智能体系统在各领域快速进步，但其评估仍然碎片化。大多数基准依赖固定的、以 LLM 为中心的线束，需要大量集成工作，造成测试-生产不匹配，且限制了不同智能体设计之间的公平比较。根本问题在于缺乏一个开放的、智能体无关的评估接口。本文倡导 Agentified Agent Assessment（AAA），其中评估由裁判智能体执行，所有参与者通过标准化协议交互：A2A 用于任务管理，MCP 用于工具访问。AAA 只需一个接口，产生了通用、统一的框架，将评估逻辑与智能体实现分离，实现可复现、可互操作和多智能体评估。
