## 📄 研究论文

- **ClinEnv: 交互式多阶段长时域电子病历环境评估 Agent** 📅 2026-06-01 | arXiv:2606.02568
  作者/机构：Yuxing Lu 等
  核心贡献：提出 ClinEnv，首个将 LLM 作为住院医师进行纵向模拟的交互式基准。每个病例自动构建为有序决策阶段序列，模型必须在每个阶段主动查询四个专业 Agent 后再做出用药、手术和诊断决策。
  摘要：临床实践不是从枚举选项中选择答案，而是医生在不确定性下逐步收集异构信息并做出不可逆的序列决策。静态基准无法探测这些能力，现有交互式医疗基准则各有妥协。ClinEnv 评估 LLM 在真实住院病例上作为主治医师的表现，同时评分"决策了什么"和"如何收集信息"。7 个模型中最强仅达 0.31 决策 F1，结果质量与过程质量严重脱钩——模型恢复出院诊断的 F1 为 0.51，但管理行动仅 0.17。ClinEnv 使仅靠结果评估无法发现的信息获取差距变得可衡量。
  链接：[arXiv](https://arxiv.org/abs/2606.02568)

- **SafeSteer: 局部策略蒸馏实现高效安全对齐** 📅 2026-06-01 | arXiv:2606.02530
  作者/机构：Hao Li, Jingkun An 等
  核心贡献：提出 SafeSteer，将安全对齐限制在安全 token 上进行策略蒸馏，而非全局权衡。仅需 100 条有害样本且无需通用数据，即可在 7 个安全基准上实现强安全表现，同时在 5 个通用能力基准上仅极小退化。
  摘要：LLM 安全对齐常导致通用能力下降（对齐税）。现有方法通过平衡双目标来缓解，严重依赖大量通用数据或辅助奖励模型。本文论证安全特征在输出分布中天然稀疏，对齐只需局部修改而非全局权衡。SafeSteer 首先通过激活引导构建安全教师，然后开发安全 token 选择算法，训练时将反向 KL 惩罚限制在这些 token 上以保留通用能力。实验表明，SafeSteer 在安全-能力权衡上显著优于现有方法，所需数据不足基线的 1%。
  链接：[arXiv](https://arxiv.org/abs/2606.02530)

- **Tracking the Behavioral Trajectories of Adapting Agents** 📅 2026-06-01 | arXiv:2606.02536
  作者/机构：Jonah Leshin 等（AIWILD Workshop @ ICML 2026）
  核心贡献：提出将 Agent 特质定义为嵌入空间方向的方法论框架，通过线性模型学习特质向量，可量化评估 Agent 行为文件的演变轨迹。在 68 对标记的差异对上对"敏感数据寻求倾向"特质达 91.2% 符号分类准确率。
  摘要：文本文件（技能文件、记忆文件、行为配置文件）在定义现代 Agent 行为中起核心作用。这些文件随人类或 Agent 自身的编辑而演化，直接影响未来交互行为。本文提出将 Agent 特质定义为文本嵌入模型嵌入空间中的方向。在标记的"前后"技能文件差异对上训练线性模型学习特质向量，然后通过将任意编辑的嵌入差异投影到该向量上来评分。该特质评估被整合到更广泛的 Agent-to-Agent 协议中，使一个 Agent 能通过可信中间人评估另一个 Agent 的技能文件更新。
  链接：[arXiv](https://arxiv.org/abs/2606.02536)

- **Bridging the Last Mile of Time Series Forecasting with LLM Agents** 📅 2026-06-01 | arXiv:2606.02497
  作者/机构：Yuhua Liao 等
  核心贡献：提出"最后一英里预测"问题——将统计预测转化为业务就绪预测的阶段，并设计 LLM Agent 框架以统一预测工作区、检索上下文证据、在结构安全约束下执行预测修订。
  摘要：时间序列预测快速发展，尤其是基础模型在数值外推上展现强零样本性能。然而在真实场景中，统计上合理的基准很少是最终预测——需结合节日效应、营销计划、外部事件等弱结构化业务语境进行修订。本文将此阶段定义为"最后一英里预测"问题，提出一个 LLM Agent 框架：维护统一预测工作区，调用工具检索上下文证据，将推理轨迹转为显式预测修订动作，支持 MapReduce 式长时域分解和事后反思记忆库。
  链接：[arXiv](https://arxiv.org/abs/2606.02497)

- **MCP-Persona: 真实个性化应用的 LLM Agent 基准** 📅 2026-06-01 | arXiv:2606.02470
  作者/机构：Wenhao Wang, Siheng Chen 等
  核心贡献：提出 MCP-Persona，首个专门评估 Agent 在真实个性化 MCP 工具上表现的基础基准，涵盖 Reddit、小红书、飞书、Slack 等社交与企业协作应用。
  摘要：MCP 已成为连接 LLM 与外部数据和工具的变革性标准，但现有基准主要关注通用信息检索工具，无法捕捉个性化社交应用中的实际挑战——工具与个人账户或本地数据库交互。MCP-Persona 涵盖多种广泛使用的应用，实验表明当前 SOTA Agent 在个性化工具使用上显著挣扎，突显了该基准的关键作用。
  链接：[arXiv](https://arxiv.org/abs/2606.02470) | [GitHub](https://github.com/wwh0411/MCP-Persona)
