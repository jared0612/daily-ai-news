## 📄 研究论文

- **Imaginative Perception Tokens Enhance Spatial Reasoning in Multimodal Language Models** 📅 2026-06-02 | arXiv:2606.03988
  作者/机构：Mahtab Bigverdi, Lindsey Li 等 / 华盛顿大学
  核心贡献：提出 Imaginative Perception Tokens (IPT)，一种让视觉语言模型"想象"未见视角的中间感知表示；构建了 PET、PT、MVC 三个空间推理任务及约 20K 数据集；证明 IPT 监督在空间推理上优于文本 Chain-of-Thought，且无需推理时生成图像。
  摘要：视觉语言模型在空间推理任务上仍存在显著不足，尤其是需要从不可见视角推断场景信息时。本文提出想象力感知令牌（IPT），作为中间感知表示，将 VLM 在替代空间配置下的感知结果外化，同时与观察输入保持一致。研究构建了视角采择（PET）、路径追踪（PT）和多视角计数（MVC）三个任务及约 2 万条标注数据。以 BAGEL 为骨干网络，IPT 在 MVC 上提升 3.4% 准确率，在 PT 上达到闭源模型水平。更关键的是，将 IPT 与仅标签监督结合可获得额外增益，而文本 CoT 反而显著降低性能，说明将空间计算强制通过语言存在模态不匹配问题。
  链接：[arXiv](https://arxiv.org/abs/2606.03988)

- **VEPO: Vision-Entropy Token Selection for Visual Reasoning RL** 📅 2026-06-02 | arXiv:2606.03937
  作者/机构：Senjie Jin, Peixin Wang 等 / 复旦大学
  核心贡献：发现传统 token 级熵在视觉推理 RL 中失效；提出 VEPO 框架，通过视觉敏感度与 token 熵的乘积耦合选择关键 token；在 7B 和 3B 规模分别超越纯熵基线 2.28 和 3.15 个百分点。
  摘要：在纯文本 RLVR 中，token 级熵被广泛用于信用分配，但本研究发现该机制在视觉推理中失效——原因是视觉敏感 token 天然具有低熵，被错误忽略。现有方法缺乏系统性视觉度量，且未能将视觉感知与语义推理交织。VEPO 通过原则性的乘积耦合机制，将梯度信用重定向到同时具有视觉基础和高信息量的 token 上。大量实验证明 VEPO 的领先性能，消融实验进一步验证了方法设计的合理性。
  链接：[arXiv](https://arxiv.org/abs/2606.03937)

- **Reasoning Structure of Large Language Models** 📅 2026-06-02 | arXiv:2606.03883
  作者/机构：Frédéric Berdoz 等 / 已被 ICML 2026 接收
  核心贡献：提出可扩展的逻辑谜题基准和推理图构建管线；将非结构化推理轨迹转化为可验证的声明-依赖图；定义推理效率指标，量化模型逻辑流的集中度。
  摘要：大型推理模型（LRM）通常仅用最终答案准确率或 token 数量评估，但相同指标可能隐藏根本不同的推理结构。本文引入可扩展的 LRM 逻辑谜题基准，以及将非结构化推理轨迹转化为可验证推理图（声明和依赖关系）的管线。推理由此成为结构化、可测量的对象，其拓扑可被定量分析。在此基础上定义推理效率指标，量化模型逻辑流的集中程度。对开源推理模型的分析表明，结构度量能区分 token 数和准确率所混淆的行为模式，是比较推理随谜题难度扩展方式的实用工具。
  链接：[arXiv](https://arxiv.org/abs/2606.03883)
