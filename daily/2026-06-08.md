## 📄 研究论文

- **How AI Agents Reshape Knowledge Work: Autonomy, Efficiency, and Scope** 📅 2026-06-05 | arXiv:2606.07489
  作者/机构：Jerry Ma 等 / Perplexity AI
  核心贡献：(1) 基于 Perplexity Search 与 Computer 产品的生产数据，首次大规模实证对比 AI Agent 与传统搜索助手在知识工作中的差异；(2) Computer Agent 每会话自主工作 26 分钟，而搜索仅 33 秒，完成时间降低 87%，成本降低 94%；(3) Agent 改变了用户尝试的工作范围，跨职业边界查询显著增加。
  摘要：前沿 AI 系统正从对话助手向自主代理转型。本研究利用 Perplexity 的 Search 和 Computer 产品数据，通过会话级自然实验揭示三大发现：第一，Computer 每用户会话执行 26 分钟自主工作，而 Search 仅 33 秒，查询不满意率降低 55%；第二，在匹配任务中完成时间从 269 分钟降至 36 分钟，时间和成本分别降低 87% 和 94%；第三，Agent 扩展了用户尝试工作的范围——跨职业边界、高阶认知、复合任务和全新工作活动显著增加。证据表明 AI Agent 不仅加速工作流，还增强输出质量、降低成本并拓展自动化工作的广度和深度。
  链接：[arXiv](https://arxiv.org/abs/2606.07489)

- **Estimating No-CoT Task-Completion Time Horizons of Frontier AI Models** 📅 2026-06-05 | arXiv:2606.07157
  作者/机构：Dewi Gould 等 / METR 等
  核心贡献：(1) 在 43 个基准、3 万+问题上系统测量前沿模型无 CoT 推理能力；(2) 定义"无 CoT 50% 任务完成时间视野"，发现前沿模型每年翻倍；(3) GPT-5.5 的无 CoT 时间视野超过 3 分钟，预测 2028 年可达 7 分钟、2030 年达 25 分钟。
  摘要：许多前沿 AI 安全监测依赖 CoT 推理监控。如果模型能在内部执行足够复杂的推理而不输出思考 token，这种监督将失效。本研究在涵盖数学、编码、谜题、因果推理、心理理论和战略推理等 43 个基准上测量前沿模型无 CoT 推理能力。定义了 50% 任务完成时间视野（TH），发现前沿模型无 CoT TH 过去 6 年大约每年翻倍，GPT-5.5 的 TH 超过 3 分钟，推理 token 视野超过 1500。中位数预测显示无 CoT TH 可能于 2028 年超过 7 分钟、2030 年超过 25 分钟。建议前沿开发者明确跟踪此指标。
  链接：[arXiv](https://arxiv.org/abs/2606.07157)

- **DuMate-DeepResearch: An Auditable Multi-Agent System with Recursive Search and Rubric-Grounded Reasoning** 📅 2026-06-05 | arXiv:2606.07299
  作者/机构：Lingyong Yan 等 / 百度 DuMate 团队
  核心贡献：(1) 提出 DuMate-DeepResearch 多代理深度研究框架，解耦 Agent Core 与工具生态；(2) 引入图动态规划、递归双层执行和 rubric 测试时优化三大机制；(3) 在 DeepResearch Bench 和 DeepResearch Bench II 上刷新 SOTA。
  摘要：深度研究（Deep Research）作为新兴代理范式面临规划、调度、幻觉和可审计性四大瓶颈。本文提出基于千帆 Agent Foundry 的 DuMate-DeepResearch 框架，将任务理解/规划/调度的 Agent Core 与检索/证据获取/报告渲染的工具生态解耦，使每个中间决策和工具调用可追溯。三大创新机制：(i) 图动态规划策略实现从粗到细的研究路线图扩展和反思/回溯/并行分支；(ii) 递归双层执行将复杂搜索子任务委托给内部 Search Agent 独立规划循环；(iii) rubric 测试时优化动态生成质量标准并作为推理脚手架。在两个深度研究基准上取得 SOTA：DeepResearch Bench 总分 58.03%，DeepResearch Bench II 总分 61.95%。
